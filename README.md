# Customer & Revenue Intelligence Dashboard

## Project Overview
This project is an interactive Tableau dashboard built to analyze customer sales performance, revenue trends, profit contribution, and regional sales distribution.

## Tools & Technologies Used
- Tableau Public
- SQL
- CSV Dataset
- Data Visualization

## Dashboard Features
- Revenue Trend Analysis
- Revenue by Country Map
- Revenue by Category
- Profit Analysis
- Top Customers by Revenue
- KPI Cards:
  - Total Revenue
  - Total Profit
  - Total Orders

## Key Business Insights
- Revenue showed a steady upward trend over time.
- Bikes category generated the highest contribution to profit and revenue.
- Certain countries contributed significantly higher sales compared to others.
- Top customers accounted for a major portion of overall revenue.
